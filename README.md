Neighbor Sum Matrix

Программа на Java, которая создаёт двумерный массив целых чисел размером N x M, заполняет его случайными значениями от 1 до 9, а затем для каждого элемента вычисляет сумму его соседей — сверху, снизу, слева и справа. Результат сохраняется в отдельный массив того же размера и выводится на экран вместе с исходным.


Что делает программа:
Пользователь вводит размеры матрицы N и M через консоль. Программа создаёт массив и заполняет его случайными числами. Затем для каждой ячейки считается сумма соседей по четырём сторонам — диагонали не учитываются. Если у элемента нет соседа с какой-либо стороны (он стоит на краю), эта сторона просто пропускается. Итоговый массив сумм выводится рядом с исходным.


Структура проекта:

Main.java — точка входа, запускает все этапы: создание, заполнение, вычисление, вывод.
Matrix.java — содержит все методы для работы с матрицей.


Как запустить:

Для запуска нужен установленный JDK. Скомпилировать и запустить можно через любую IDE (например, IntelliJ IDEA) и запустить Main.java
После запуска программа попросит ввести N (количество строк) и M (количество столбцов).
